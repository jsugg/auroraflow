## Maintenance rules

- Keep one row per `AUR-DEC-*` decision.
- Update this file before a behavior-changing PR relies on a changed decision.
- Link follow-up work with `AUR-ARCH-*`, `AUR-IMPL-*`, and validation IDs where applicable.
- Record implementation progress and validation output in the implementation journal.
- Revisit decisions only with new evidence listed in the table; do not silently broaden scope.
